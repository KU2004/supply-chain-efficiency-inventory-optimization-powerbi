# 📊 Supply Chain Efficiency & Inventory Optimization Dashboard | Power BI

## 🚀 Project Overview
This project focuses on analyzing supply chain performance and optimizing inventory management using an interactive Power BI dashboard.

The dashboard helps track inventory levels, supplier performance, delivery efficiency, and demand forecasting to support data-driven decision-making.

---

## 🎯 Objective
- Monitor inventory levels across products and warehouses
- Identify stockout risks and reorder requirements
- Evaluate supplier delivery performance
- Compare forecast demand vs actual demand
- Improve overall supply chain efficiency

---

## 🛠️ Tools & Technologies Used
- Power BI (Data Visualization & Dashboard)
- Excel / CSV (Dataset)
- DAX (Data Analysis Expressions)
- Power Query (Data Transformation)

---

## 📂 Dataset Description
The dataset includes:
- Product details (ID, Name, Category)
- Inventory data (Opening Stock, Incoming Stock, Closing Stock)
- Sales data (Units Sold)
- Supplier information
- Warehouse & region data
- Delivery performance (On-Time / Delayed)
- Demand forecasting (Forecast vs Actual)
- KPIs like Inventory Turnover, Forecast Accuracy, Holding Cost

---

## 🔄 Data Preparation (Power Query)
- Removed null values and duplicates
- Standardized product, supplier, and location names
- Converted data types (date, numeric, percentage)
- Created Month, Quarter, and Year columns
- Added calculated columns like Reorder Status

---

## 🧮 Key DAX Measures
- Total Inventory
- Total Units Sold
- Stockout Count
- Reorder Count
- Average Lead Time
- On-Time Delivery %
- Forecast Accuracy %
- Inventory Holding Cost
- Inventory Turnover Ratio

---

## 📊 Dashboard Features

### 🔝 KPI Cards
- Total Inventory
- Units Sold
- Stockout Count
- On-Time Delivery %
- Forecast Accuracy %
- Inventory Turnover Ratio

### 📈 Visuals
- Inventory by Product Category (Column Chart)
- Supplier Delivery Performance (Bar Chart)
- Monthly Sales Trend (Line Chart)
- Delivery Status Distribution (Donut Chart)
- Forecast vs Actual Demand (Clustered Column Chart)
- Detailed Product Table (Matrix)

### 🎛️ Filters / Slicers
- Region
- Supplier
- Product Category
- Warehouse Location
- Month

---

## 💡 Key Insights
- Identified products with high stockout risk
- Highlighted suppliers with frequent delivery delays
- Analyzed demand forecast accuracy
- Compared performance across regions and categories
- Improved visibility into inventory turnover and holding costs
